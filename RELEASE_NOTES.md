# 팔랑북 릴리즈 노트

## 1.0.1 (2026-07-20)

### 새 기능
- **도장 크기 조절**: 도장 팔레트 아래에 크기 점 3개(작게·보통·크게)가 생겼어요. 붓 크기와 따로 조절할 수 있어요.
- **호버 미리보기**: Apple Pencil을 화면 가까이 가져가거나(호버 지원 기기) 트랙패드 포인터를 올리면, 그리기 전에 도장이 찍힐 자리와 모양, 붓·지우개가 닿을 크기가 반투명하게 미리 보여요.

### 개선
- **길게 녹음하기**: 마이크 녹음 중에는 애니메이션이 계속 반복 재생돼요. 원하는 만큼 목소리를 입히고 정지 버튼을 누르면 끝나요. 길게 녹음한 목소리는 재생할 때 끊기지 않고 끝까지 나와요.

## 1.0 (2026-07-20)

### 새 기능
- **스탬프(도장) 기능**: 그리기 화면 양옆의 빈 공간이 도장 팔레트가 되었어요. 하트·동그라미·네모·세모·별·꽃·자동차·새 8가지 도장을 골라 캔버스를 탭하면 그 자리에 찍혀요. 색은 색 팔레트, 크기는 S/M/L 붓 크기를 그대로 따라가고, 찍은 도장은 지우개·되돌리기·재생·내보내기에서 그린 그림과 똑같이 동작해요.
- **한번에 다 지우기**: 휴지통 버튼으로 현재 페이지의 그림을 한 번에 지울 수 있어요. 실수로 눌러도 되돌리기 버튼 한 번이면 페이지 전체가 복원돼요.
- **빈 화면에서 시작하기**: 홈 화면의 점선 테두리 하얀 종이 카드를 누르면 동영상 없이 빈 페이지만 있는 새 프로젝트가 만들어져요. 그리다가 "다음" 버튼을 누르면 페이지가 계속 늘어나요.

### 개선
- **스크롤 없는 홈 화면**: 만들기 버튼·따라 그리기·저장된 프로젝트가 모두 화면을 꽉 채우는 한 그리드에 정리됐어요. 프로젝트가 많아져 화면을 넘칠 때만 스크롤이 생겨요.
- **두 줄 고정 툴바**: 그리기 화면 상단 툴바가 가로 스크롤 없이 두 줄로 항상 전부 보여요. 윗줄은 도구, 아랫줄은 색과 붓 크기예요.
- **하단 재생 바 숨기기**: 재생 바 오른쪽 ⌄ 버튼으로 바를 내려 캔버스를 넓게 쓸 수 있어요. 오른쪽 아래 ⌃ 버튼으로 언제든 다시 올릴 수 있어요.

### 이전 업데이트
